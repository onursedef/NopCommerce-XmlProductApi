# XML API Plugin for NopCommerce

This plugin allows you to access the list of your products in XML format. It is useful for integration with external systems.

## Installation

1. Clone the repository to Plugin folder in your NopCommerce solution.
2. Add the project to your solution.
3. Build the solution.
4. Run the application.

## Usage

1. Go to the Admin area.
2. Go to Configuration -> Local Plugins -> xmlApi and click on the Install button.
3. Restart the application.

Now you can see the product details in XML format by going to the following URL: 
```
https://<your-nopcommerce-store>/Api/Products
```
